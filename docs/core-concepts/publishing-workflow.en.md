# Publishing workflow

Use this sequence when the public docs outline changes:

1. Update or add source pages under `docs/**`.
2. Update the English and Chinese navigation manifests.
3. Keep links to legacy service docs stable.
4. Reload `docs.svc.plus` and verify grouped navigation plus page fallbacks.
