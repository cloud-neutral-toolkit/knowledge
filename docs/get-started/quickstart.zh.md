# 快速开始

如果你需要快速进入这套公开文档，按下面顺序阅读即可。

## 面向读者

1. 从 [欢迎页](/docs/get-started/overview) 开始。
2. 再看 [基础认知](/docs/core-concepts/overview)。
3. 需要规范或迁移说明时进入 [参考资料](/docs/reference/overview)。

## 面向维护者

1. 所有公开文档都在 `docs/**` 下维护。
2. 双语入口页应成对存在，并带 `lang: en` / `lang: zh`。
3. 导航结构变化时同步更新 `docs/navigation.en.yaml` 与 `docs/navigation.zh.yaml`。
4. 历史服务文档优先链接归位，不要先复制出第二份。
