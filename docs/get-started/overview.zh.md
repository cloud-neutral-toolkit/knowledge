# 欢迎页

`docs/` 是 **`docs.svc.plus`** 的规范入口。

这个入口页只回答三件事：公开文档放在哪里、按什么方式分组、接下来应该读什么。

## 这套文档覆盖什么

- `docs/**` 下的仓库级文档
- `docs.svc.plus` 的分组导航
- 对历史服务文档的稳定引用

## 这套文档不覆盖什么

- `content/**` 下的博客与随笔内容
- `blogs.svc.plus` 的内容发布流

## 建议下一步

1. 先看 [快速开始](/docs/get-started/quickstart)。
2. 再看 [项目地图](/docs/get-started/project-map)。
3. 然后进入 [信息架构](/docs/core-concepts/information-architecture)。
